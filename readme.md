
## Back-end API

Restful API simples escrita em php usando Laravel 5.7, que permite a realização de operações básicas de CRUD

## Setup
```
composer install
php artisan migrate
php artisan db:seed
php artisan key:generate
```
## Endpoints
### GET 
``` 
api/products
```
### POST 
```
api/products/
```
### PUT 
```
api/products/{id}
```
### DELETE
```
api/products/{id}
```
